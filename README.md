# Marwan Bahgat — Portfolio

Personal portfolio website built with React, Vite, TypeScript, Tailwind CSS v4, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build for Production

```bash
npm run build
```

Output is in the `dist/` folder.

## Preview Production Build

```bash
npm run preview
```

## Deploy

### Vercel
Import the repo on [vercel.com](https://vercel.com). `vercel.json` is already configured for SPA routing.

### Netlify
Import the repo on [netlify.com](https://netlify.com). `netlify.toml` is already configured for SPA routing.

Build command: `npm run build`  
Publish directory: `dist`
